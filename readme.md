# AquaPiece - Puzzle Game

## 🎮 ゲーム概要
AquaPiece は、画面上に散らばったロゴ（ピース）を **ドラッグ＆ドロップ** で正しい位置に配置し、ステージをクリアしていくパズルゲームです。各ステージは異なる配置パターンとタイムチャレンジが用意されており、クリアタイムと正確さでスコアが評価されます。

## 🥅 目的
- **全ピースを正しく配置**: すべてのロゴが `placed` 状態になるとステージクリア。
- **最速タイムでクリア**: クリアタイムが短いほどランキング上位に。
- **図鑑を完成させる**: クリアしたピースは図鑑に登録され、全種集めが最終目標。

## 🕹️ 遊び方
1. **スタート画面**: `START GAME` ボタンをクリックしてゲーム開始。
2. **ステージ選択**: `Stage 1`, `Stage 2`, `Stage 3` のいずれかを選択。
3. **ピース操作**:
   - **左クリック**: ピースをドラッグして移動。
   - **右クリック**: ピースを回転（90°単位）。
   - 正しいマスにドロップすると自動で `placed` 状態に切り替わります。
4. **クリア確認**: 全ピースが `placed` になると `CLEAR!` オーバーレイが表示され、クリアタイムとステージ別ランキングが表示されます。
5. **図鑑閲覧**: 画面右上の `図鑑` ボタンでロック中・取得済みのピースを確認できます。
6. **ランキング閲覧**: `ランキング` ボタンで各ステージのベストタイム一覧が表示されます。

## ⌨️ 操作方法 (キーボード/マウス)
- **左クリック**: ピースの移動（ドラッグ）
- **右クリック**: ピースの回転
- **UI ボタン**: ステージ選択、図鑑・ランキング表示、戻るボタン

## 📊 スコアリング & ランキング
- **タイム表示**: 画面左上の `時間: mm:ss` がリアルタイムで更新されます。
- **クリアランキング**: 各ステージのベストタイムは `ranking-list-1`, `ranking-list-2`, `ranking-list-3` に保存され、次回起動時に閲覧可能です。
- **保存方法**: `localStorage` を利用してブラウザにデータを保持します。

## 🛠️ 開発メモ
- **ドラッグ実装**: `logo-piece` クラスに `grab` カーソルと `pointer-events: none` を設定し、タッチデバイスでも操作可能にしています。
- **ステージロジック**: `startStage(stageId)` でステージデータをロードし、全ピースが `placed` になると `ranking-list` を更新します。
- **エフェクト**: `bubbleCanvas` が背景アニメーション、`wave-transition` がステージ切替エフェクトとして使用されています。
